# /machina-log Stand-to Template

**Use in #legio-machinatorum channel daily at 09:00 UTC.** Each Tribunus/agent posts a single block following this structure.

```
/machina-log
Agent: <Callsign / AG-ID>
Mission: <campaign or kit>
Status: 🟢 / 🟡 / 🔴
Since last: <concise wins>
Next: <next deliverables + ETA>
Blockers: <issue + owner> (omit if none)
Spend: <£ + source> (only if changed)
Notes: <links, decisions, asks>
```

**Guidance**
- 🟢 on track, 🟡 risk, 🔴 blocked.
- Keep “Since last / Next” to bullet fragments (max 3 lines each).
- Escalations also ping Forge Marshal if 🔴.
- Quaestor appends a summary reply with budget delta + approvals.

Copy this template verbatim when setting up the slash command or automation.
