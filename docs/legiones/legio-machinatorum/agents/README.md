# Legio Machinatorum – Agent Dossiers

Each agent card lives in this directory under `AG-XXXX.md`. Use the following template:

```
# Agent <Callsign> (AG-XXXX)

**Mandate:**
- 

**Definition of Done:**
- 

**Inputs:**
- Triggers or artefacts required to engage.

**Outputs:**
- Deliverables + recipient cohorts/legions.

**Cadence & Rituals:**
- Stand-ups, reviews, syncs this agent must attend/host.

**Handoffs & Escalations:**
- Who to notify when blockers hit redline and what thresholds trip an escalation.

**Tooling & Access:**
- Repos, platforms, credentials (reference secrets vault IDs, never inline secrets).

**Open Threads / TODOs:**
- Outstanding work, risks, decisions pending.
```
