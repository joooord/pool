# Data Request Template

```
Subject: <Business/Pillar> intel needed for <project>

Context:
- Objective / capability unlocked
- Deadline or cadence

Requested Items:
1. <Item> — reason, preferred format, source (dashboard, export, login)
2. <Item>

Access & Constraints:
- Current tools/logins we have
- Missing permissions or approvals needed

Delivery:
- Drop files in <path> or share link
- Note sensitivity levels (public/internal/confidential)

Next Steps:
- What I’ll build once data arrives
```

Adapt tone to WhatsApp/Slack/email as needed. Always tie requests to tangible outcomes.
